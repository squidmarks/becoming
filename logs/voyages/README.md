# Voyage Logs

This directory contains records of voyages, passages, and significant events aboard M/Y Becoming.

## Format

Each voyage or significant event can be documented as:
- Individual markdown files (e.g., `2026-03-charleston-to-beaufort.md`)
- Git commits with detailed messages
- Combination of both

## Log Entry Template

```markdown
# [Date] - [Voyage/Event Title]

**Date:** YYYY-MM-DD  
**Departure:** Location  
**Arrival:** Location  
**Distance:** XX nm  
**Duration:** XX hours  
**Crew:** Names  

## Weather
- Conditions
- Wind
- Sea state

## Notes
- Events
- Systems performance
- Maintenance performed

## Systems Status
- Inverter/batteries
- Engines
- Navigation electronics
- Any issues or observations
```
