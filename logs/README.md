# Ship's Log - M/Y Becoming

This directory contains chronological records of all significant events aboard *Becoming*.

## Organization

Logs are organized by type:

- **`voyages/`** - Voyage logs, passages, sea trials
- **`maintenance/`** - Maintenance activities, repairs, routine service
- **upgrades/`** - System upgrades, modifications, installations

## Naming Convention

Use the format: `YYYY-MM-brief-description.md`

Examples:
- `2026-03-charleston-to-beaufort.md`
- `2026-02-engine-oil-change.md`
- `2025-09-lipo-battery-installation.md`

## Log Entry Template

```markdown
# [Date] - [Title]

**Date:** YYYY-MM-DD  
**Type:** Voyage | Maintenance | Upgrade  
**Location:** [Where work was done / departure-arrival]  
**Crew/Personnel:** [Who was involved]  

## Summary
Brief overview of the event/work

## Details
Detailed description, observations, decisions made

## Systems Affected
- List of systems touched (link to systems/ docs if applicable)

## Parts/Materials Used
- Any parts, materials, or equipment

## Cost
- Labor and materials cost (if applicable)

## Follow-up Required
- Any tasks or monitoring needed

## Photos/Diagrams
- References to images (store in this directory or systems/)
```

## Cross-References

Log entries should reference:
- **System docs** (`systems/`) for technical details
- **Procedures** (`docs/procedures/`) for work performed
- **Manuals** (`docs/manuals/`) for equipment references

---

*The log is the memory of the ship. Keep it detailed and current.*
