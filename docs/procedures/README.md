# Operating Procedures

Standard operating procedures, checklists, and how-to guides for vessel operations.

## Categories

- **Daily Operations** - Normal underway and docking procedures
- **Emergency Procedures** - What to do in emergencies
- **Seasonal** - Winterization, commissioning
- **Maintenance** - How to perform routine maintenance tasks

## Format

Each procedure should include:
- Clear step-by-step instructions
- Safety warnings
- Required tools/materials
- Expected duration
- References to system documentation

---

*Safe operations require documented, practiced procedures.*
